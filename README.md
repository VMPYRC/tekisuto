## Use STBL

1. Go to [releases](https://github.com/VMPYRC/tekisuto/releases)
2. Download and extract the .ZIP file
3. Move the folder to `Documents\Electronic Arts\The Sims 3\Mods`
4. Can be placed in `Packages` OR `Overrides`
5. Play + Enjoy
